# Product Overview

A self-hosted AI coding agent that runs on a remote desktop machine with local GPU acceleration. The system provides an alternative to cloud-based coding assistants by running entirely offline using open-source LLMs.

The agent integrates with VSCode through a custom extension and provides intelligent code assistance, multi-file editing, and task execution capabilities. It's designed for developers who need AI coding assistance while maintaining full control over their code and infrastructure.

Target hardware: NVIDIA RTX 3080 (10GB VRAM), 32GB RAM, 6+ core CPU.
