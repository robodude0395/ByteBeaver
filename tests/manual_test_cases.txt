================================================================================
                    BYTEBEAVER AGENT - MANUAL TEST CASES
================================================================================

This document contains manual test prompts to evaluate the AI agent's
capabilities across code generation, workspace interaction, tool usage,
and conversational abilities.

================================================================================
SECTION 1: CODE GENERATION - STANDALONE PROJECTS
================================================================================

TEST 1.1: Fibonacci Sequence
PROMPT: "Create a Python script that generates the first 20 Fibonacci numbers"
EXPECTED OUTPUT:
- Agent uses write_file tool to create a .py file
- Script runs without errors
- Output shows correct Fibonacci sequence: 0, 1, 1, 2, 3, 5, 8, 13...

TEST 1.2: Snake Game
PROMPT: "Build a simple snake game in Python using curses"
EXPECTED OUTPUT:
- Agent creates a playable snake game file
- Game has basic controls (arrow keys or WASD)
- Snake grows when eating food
- Game ends on collision with wall or self

TEST 1.3: Tetris Clone
PROMPT: "Create a basic Tetris game in Python"
EXPECTED OUTPUT:
- Agent creates a working Tetris implementation
- Pieces fall and can be rotated
- Lines clear when complete
- Game over detection works

TEST 1.4: Todo List CLI
PROMPT: "Make a command-line todo list app in Python with add, remove, list, and mark-complete features"
EXPECTED OUTPUT:
- Agent creates a functional CLI app
- All CRUD operations work
- Data persists between runs (file-based storage)

TEST 1.5: REST API Server
PROMPT: "Create a simple FastAPI server with CRUD endpoints for a 'notes' resource"
EXPECTED OUTPUT:
- Agent creates a FastAPI app with proper endpoints
- Includes GET, POST, PUT, DELETE routes
- Uses Pydantic models for validation

TEST 1.6: Calculator with GUI
PROMPT: "Build a calculator with a graphical interface using tkinter"
EXPECTED OUTPUT:
- Agent creates a working GUI calculator
- Basic operations (+, -, *, /) work
- Clear button functions correctly

================================================================================
SCORING GUIDE
================================================================================

For each test, rate on a scale of 1-5:

1 = Complete failure (wrong output, crash, or refusal when should work)
2 = Partial failure (some correct behavior but major issues)
3 = Acceptable (works but with minor issues or inefficiencies)
4 = Good (correct output, appropriate tool usage)
5 = Excellent (optimal solution, great UX, efficient tool usage)

Key metrics to track:
- Tool usage accuracy (uses right tool for the job)
- Code quality (generated code runs without errors)
- Response relevance (answers the actual question)
- Conversation flow (natural, not robotic)
- Error handling (graceful failures, helpful messages)
- Efficiency (minimal tool calls, no unnecessary steps)
